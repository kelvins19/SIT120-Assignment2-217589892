Purpose of this web application:

  • To be a companion app with the Pokemon Sword and Pokemon Shield game.
  • This web application is targeted for the player of either or both of those game.
  • It can also be accessed by everyone, even though they are not a player of the game.
  • User can see news of the event or update of the game.
  • User can see the leaderboard of the global player.

This Project requires this component:

  • Form Input Binding
  • Form Validation
  • Transition
  • Binding Inline Styles With Object Syntax
  • CSS Hover Selector
  • HTML Tables and CSS Tables

How the Web will navigate with each other page:

From the flowchart below, we can see that each web page has menu bar, that contains all the other 5 pages links. In home page, it has a content slider that contain event, update, and leaderboard hyperlinks. In event and update page, it has three boxes that has description of the corresponding box, when the mouse hover to its box. In leaderboard page, there is a scrollable table, that lists all player that play the games. In login page, there is a form field that asked for username and password. The form will be validated first before get submitted, when the user clicks the submit button.
